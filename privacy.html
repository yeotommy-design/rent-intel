Privacy

RentIntel should collect only what is needed to run a useful public product.

The current launch build is designed to stay lightweight. RentIntel should use privacy-conscious analytics, optional feedback forms, and optional account tools only when they help improve public rent checks, saved tools, or future alerts.

Search data: may be used in aggregate to improve area matching, coverage review, and product quality.

Contact requests: should only be used to reply to corrections, partnerships, or product feedback that you asked us to review.

Analytics: should stay lightweight and focused on understanding which public pages and tools are useful.

Optional account tools: if saved areas or alerts are added later, they should collect only the minimum information needed to run them.
